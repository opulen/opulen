
# Opulen Icons

React Icons library for Opulen UI library


## Installation

Install Splendid React Icons

```bash
  npm install @opulen/icons
  bun add @opulen/icons
```
    
## Usage/Examples

```javascript
import { IconUserFilled } from "@opulen/icons"

function App() {
  return <IconUserFilled />
}
```
